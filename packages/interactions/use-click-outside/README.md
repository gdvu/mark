<p align="center">
`useClickOutside` is a custom hook for listening clicks outside of a specific element.
</p>

## Install

```bash
npm i @raddix/use-click-outside
# or
yarn add @raddix/use-click-outside
# or
pnpm add @raddix/use-click-outside
```
