## Instalación

Instale el custom hook desde su terminal.

<Snippet pkg text='@raddix/use-boolean' />

## Uso

En este ejemplo, vamos a abrir un menu cuando el cursor pase sobre el botón y se
cerrará cuando el cursor salga del contenedor del menú. Si hace click sobre el botón
se abrira/cerrará.

```tsx
import { useBoolean } from '@raddix/use-boolean';
import { Menu } from './menu'

export default function App() {
  const [open, setOpen] = useBoolean();

  return (
    <div onMouseLeave={setOpen.off}>
     <button onMouseEnter={setOpen.on} onClick={setOpen.toggle}>
       Menu
     </button>

      <Menu />
    </div>
  )
}
```
