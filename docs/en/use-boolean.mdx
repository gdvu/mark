## Usage

In this example, we will open a menu when the cursor hovers over the button and it will
close when the cursor leaves the menu container. If you click on the button, it will
open/close.

```tsx
import { useBoolean } from '@raddix/use-boolean';
import { Menu } from './menu'

export default function App() {
  const [open, setOpen] = useBoolean();

  return (
    <div onMouseLeave={setOpen.off}>
     <button onMouseEnter={setOpen.on} onClick={setOpen.toggle}>
       Menu
     </button>

      <Menu />
    </div>
  )
}
```
